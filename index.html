What is IHaveNoInternet?

IhaveNoInternet is an SMS-based navigation app, built using the Nexmo SMS API and Google Cloud Platform's Google Maps APIs.
Have you ever run out of data while in an unfamiliar place and needed to get home? Ever found yourself without wifi or reliable internet but with phone service good enough to send and receive texts? Using our app you can text a location that you would like to go to and it will return Google Maps' directions to your phone via SMS - no internet required!
Soon we'll be adding even more features! - pre-set location shortcuts, find your friend, access Google Search, and more.
